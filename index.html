Mina git-kommando!

Specifika git-kommando är:

  • git status : kollar av om filer är sparade + commitade
  • git log : visar gjorda comitter
  • git add . : Vi sparar nya ändringar
  • git commit -m "nytt meddelande" : Vi sätter id på det vi sparade
  • git push : Vi skickar våran sparning/commit till cloud-servern
  • git pull : Hämtar fjärr-repot

Allmänna terminal-kommando!

  • cd ( change directory ) flyttar till en mapp
  • pwd ( print working directory ) visar i CLI vart du är just nu
  • ls ( list ) visar innehåll i en mapp
  • mkdir ( make directory ) skapar en mapp
  • touch skapar en tom ny fil med en angiven namn
  • rm ( remove ) raderar en mapp eller fil
  • rmdir ( remove directory ) raderar en mapp
  • rm –rf Mappnamn raderar en mapp som innehåller filer
  • mv ( move ) flyttar en mapp eller fil
  • cp ( copy ) kopierar en mapp eller fil
